# EpochCoverage — Compliance, Audit & Risk

EpochCoverage wraps the EpochCore platform in a continuous compliance and risk telemetry layer, built for regulated deployments.

## Primitives

- **SBOMs** — CycloneDX 1.5 + SPDX 2.3 hybrid bills of materials per release.
- **Quantum watermark** — tamper-evident binding for every agent response and pixel tensor.
- **Audit trail** — signed, append-only execution ledger.
- **Risk telemetry** — real-time policy violation signals.

## Frameworks Addressed

- NIST AI RMF
- ISO/IEC 42001
- SOC 2 audit evidence
- Sector-specific (finance, healthcare)

## Outputs

- Continuous compliance dashboards
- Exportable audit packages
- Watermark verification endpoints
