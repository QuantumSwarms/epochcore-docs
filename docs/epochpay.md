# EpochPay — Programmable Settlement

EpochPay is the settlement layer for agent-native commerce. It enables deterministic, auditable value exchange between agents, humans, and enterprise systems.

## Features

- Stripe + on-chain settlement bridges
- Agent-to-agent micro-transactions
- Policy-bound disbursements
- Signed receipts anchored to the quantum watermark

## Use Cases

- Usage-metered agent APIs
- Marketplace-scale tool routing
- Compliance-anchored payouts

## Integrations

- Stripe
- EVM-compatible chains (via MetaMask / Uniswap flows)
- Enterprise ERP connectors
