# Platform Overview

EpochCore is a unified platform for deploying quantum-anchored, PQC-safe AI agents at enterprise scale.

## Architecture Layers

1. **Substrate** — D-KaP knowledge tensors, quantum randomness, and PQC key material.
2. **Runtime** — EpochOS orchestrates swarm-scale agent execution through a single API surface.
3. **Economy** — EpochPay handles programmable settlement for agent transactions.
4. **Governance** — EpochCoverage provides compliance, audit, and risk telemetry.
5. **Fabric** — QuantumSwarms is the agent fabric binding everything together.
